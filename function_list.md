### 画面表示機能

- **ホーム画面表示**: アプリの初期画面を表示します。
    - [ ]  トレーナー（泉さん）の写真を表示する機能
    - [ ]  目標回数（10or20or30回）を選択するUI（リスト等）を表示する機能
    - [ ]  トレーニング開始ボタンを表示する機能
- **トレーニング画面表示**: トレーニング中の画面を表示します。
    - [ ]  スマートフォンのインカメラ映像をリアルタイムで表示する機能
    - [ ]  トレーナーの写真を画面隅に表示する機能
    - [ ]  現在のカウント数を表示する機能
    - [ ]  リタイアボタンを表示する機能

### ユーザー操作機能

- [ ]  **回数設定機能**: ユーザーが目標回数を設定できるようにする機能。
- [ ]  **トレーニング開始機能**: 「開始」ボタンが押されたことを検知し、トレーニング画面へ遷移させる機能。
- [ ]  **トレーニング中断機能**: 「リタイア」ボタンが押されたことを検知し、トレーニングを終了してホーム画面へ戻す機能。

### コアロジック機能

- [ ]  **カメラ制御機能**: アプリケーションからスマートフォンのインカメラを起動・停止する機能。
- [ ]  **運動判定（顔認識）機能**: カメラ映像をリアルタイムで解析し、ユーザーの脚から顔が出てきて顔認識されたタイミングで腹筋運動が1回行われたかを判定する**本アプリの最重要機能**。
- [ ]  **カウント管理機能**: 運動判定に応じて現在の回数を加算し、目標回数に達したかを判定する機能。

### 音声再生機能

- [ ]  **イベント別音声再生機能**: 特定のタイミングで、対応する音声を再生する機能。
    - **開始時**: 「今日もはりきっていきましょう！」
    - **カウント毎**: 「1！」「2！」...
    - **中間地点**: 「あと半分！」
    - **ラストスパート**: 「あと5回！」
    - **目標達成時**: 「よく頑張りました」
    - **リタイア時**: 「次は頑張りましょう」

### データ管理機能 (バックエンド)

- [ ]  **トレーナー情報提供機能**: フロントエンドからのリクエストに応じ、データベースに保存されたトレーナーの写真URLと各種音声ファイルのURLを提供するAPI機能。
